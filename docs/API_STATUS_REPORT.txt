六、重要技术说明
================================================================================

1. 认证方式
   - 仅支持邮箱注册/登录和短信验证码登录
   - 不支持 Google OAuth 和 Apple Sign-In
   - 验证码发送字段为 address（非 email）
   - 响应字段为 iotCaptchaId（非 captchaId）

2. 请求签名
   - 使用 IoT 签名认证
   - 请求头需要: IOT-AppID, IOT-AppSecret, IOT-Token
   - IOT-Open-Body-Hash: MD5(body + timestamp + nonce)
   - IOT-Timestamp: 当前时间戳（秒）
   - IOT-Nonce: 随机字符串

3. 常见属性 key
   - soc (电池电量 %)
   - batteryPower (电池功率 W)
   - acPower (AC 输出功率 W)
   - solarPower (太阳能输入功率 W)
   - outputPower (输出功率 W)
   - batteryTemp (电池温度 °C)
   - workMode (工作模式: 0=正常, 1=备份, 2=节能)
   - acOut1Enable, acOut2Enable (AC 输出开关)
   - usbOut1Enable (USB 输出开关)
   - sleepMode (睡眠模式)

4. 设计系统
   - Primary: #01D6BE → 已更换为 #0D9488
   - Light: #00E5CA → 已更换为 #14B8A6
   - Background: #000000
   - Card: #1C1C1E
   - 圆角: sm=8px, md=14px, lg=20px, xl=28px
